# `gpic`

Downloads a Google account's profile picture.

## Install

```bash
cd \
  gpic
make \
  DESTDIR=/ \
  PREFIX=/usr \
  install
```

## Usage

```bash
gpic \
  <email> \
  (<output_file>)
```
